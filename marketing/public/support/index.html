Get in touch

The fastest way to reach a human is email: contact@curlbro.com. CurlBro is built and maintained by one person, so replies usually take a couple of days.

Questions, suggestions, app-store reviews you'd like addressed, partnership inquiries — all welcome at the same address.

Report a bug

Found something broken? The best place to file it is GitHub: github.com/SpacemanSpiff7/CurlBro/issues. You can also use the Report a Bug button inside the app (Settings → Help) — it pre-fills your browser, screen size, and platform so we can reproduce faster.

A useful bug report includes:

  • What you were trying to do
  • What happened instead
  • Steps to reproduce, if you can repeat it
  • Browser / device / OS
  • A screenshot or short screen recording, if relevant

If you'd rather not use GitHub, email is fine — the same details just go in the email body.

Frequently asked

My workouts disappeared.

CurlBro stores everything in your browser's local storage on the device you're using. It's not synced to a server. Clearing your browser's site data, switching to private / incognito mode, or using a different browser or device will all show an empty library. If you exported your workouts before, you can import them back from Settings → Data.

Sharing or "Copy to clipboard" isn't working.

The browser's clipboard API only works over HTTPS. If you're using http:// (some local network setups), copy-to-clipboard will silently fail. Use the share/export options instead, or open the app at https://curlbro.com.

The screen zooms in when I tap an input on iPhone.

iOS Safari auto-zooms text inputs smaller than 16px and never resets after. CurlBro's inputs are tuned to avoid this; if you're seeing it, you're likely on an older cached version — pull down to refresh, or close the tab and reopen the page.

Can I sync across devices?

Not automatically — there are no accounts and no server. To move data between devices, use Settings → Export on the source device and Settings → Import on the destination.

Is there an iOS app?

Yes — the iOS app stores data locally with Apple Health integration. The web app is fully featured and doesn't require an install. Both are independent; data does not sync between them.

How do I delete all my data?

Settings → Data → Clear All Data. Or, in your browser, clear site data for curlbro.com. CurlBro never receives a copy, so there's nothing on our end to delete.

Privacy and data

CurlBro stores all your workout data locally and does not collect personal information. Full details, including how Apple Health and exports work, are in the Privacy Policy.

Feature requests

Have an idea? Open an issue on GitHub with the enhancement label, or email contact@curlbro.com. Real use cases ("I'm doing X and the app makes me do Y workaround") help much more than abstract feature names.
